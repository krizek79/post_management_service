#   Post Management Service
##  Used Technologies
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL Database
- OpenAPI 3.0
- Docker
##  Installation
Maven:&nbsp;`mvn clean install -DskipTests`\
Docker: `docker-compose up -d`
